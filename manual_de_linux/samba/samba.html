Manual de Carpetas Compartidas SAMBA desde Windows

1. Antes de empezar (Relleno)

Recomiendo un ubuntu 22.04 o debian 12
y Tener derechos de admin (su o sudo su)

1.2 Demonios de SAMBA

  • smbd: Servicio principal que maneja la autenticación de usuarios y comparte recursos.
  • nmbd: Servidor de nombres de Windows que actúa como cliente y servidor de nombres WINS.

1.3 Utilidades de SAMBA

  • smbclient: Cliente estilo FTP.
  • smbtar: Programa equivalente al comando Unix "tar".
  • nmblookup: Realiza búsquedas de nombres NetBIOS sobre TCP/IP.
  • smbpasswd: Administrador de contraseñas.
  • smbstatus: Reporta las conexiones de red actuales hacia los recursos compartidos por el servidor Samba.
  • testparm: Verifica la sintaxis de /etc/samba/smb.conf.
  • testprns: Verifica las impresoras compartidas de SAMBA.

2. Instalación y Configuración de SAMBA

2.1 Instalación de SAMBA

Ejecutamos el siguiente comando para instalar el servicio SAMBA:

apt-get install samba

2.2 Reinicio del Servicio SAMBA (Esto se hace para aplicar las configuraciones)

Dependiendo de la implementación del servicio, se puede reiniciar SAMBA usando:

systemctl restart smbd

O

service smbd restart

Nota: Esto depende de la distribución de linux que se llegue a usar.

3. Compartición Sencilla de SAMBA. (Edicion Fichero /etc/samba/smb.conf)

3.1 Crear Directorio y Asignar Permisos en Linux

3.1.1 Crear Directorio

mkdir -p /compartidosamba

3.1.2 Asignar Permisos a la carpeta local.

Escenarios comunes:

Nota: Los permisos de Linux prevalecen sobre los permisos de SAMBA.
Nota 2: si usais 777 sirve de lujo abierto para to cristo. (chmod 777)

a. Permisos de escritura para todos los usuarios: (yo uso este que funciona)

chmod o+w /compartidosamba

Verificar permisos:

ls -al /compartidosamba/
drwxr-xrwx 2 root root 4096 oct 16 13:44

b. Directorio exclusivo para un usuario:

chown pepito /compartidosamba/

Verificar permisos:

ls -al /compartidosamba/
drwxr-xr-x 35 pepito pepito 4096 oct 16 11:19 pepito

c. Directorio para un grupo de usuarios:

groupadd samba
chgrp samba /compartidosamba2/
chmod g+w /compartidosamba2/
adduser pepito samba

Verificar permisos:

ls -al /compartidosamba2/
drwxrwxr-x 2 root samba 4096 oct 16 13:02 .

3.2 Añadir Usuario SAMBA

Nota: tenemos que crearlo primero usando 

adduser fernando


smbpasswd -a fernando

3.3 Establecimiento de Opciones para Compartir

Editar el archivo de configuración:

nano /etc/samba/smb.conf

Parámetros básicos:

  • [nombre_de_recurso]: Nombre del recurso a conectar.
  • browseable: Si el nombre del recurso es visible desde la red.
  • path: Directorio compartido.
  • writable: Habilita escritura para el recurso.
  • valid users: Lista de usuarios con acceso permitido.
  • guest ok: Permite acceso a usuarios anónimos.

Ejemplo de configuración:

[smbcompartido]
	comment = Compartido samba
	path = /compartidosamba
	valid users = fernando
	writable = yes
	browseable = yes
	guest ok = no


Para compartir sin limitar usuarios:

[COMPARTIDO2]
	path = /compartidosamba2/
	writeable = yes
	browseable = yes

3.4 Reiniciar el Servidor SAMBA (Aplicar configuración)

/etc/init.d/samba restart

o

systemctl restart smbd

4. Acceso a la Carpeta Compartida

4.1 Acceso desde Linux

4.1.1 Línea de Comandos

Crear punto de montaje:

mkdir -p /mnt/mipuntodemontaje

Conectar carpeta compartida:

mount -t cifs -o username=pepito,password=pepe2000 //192.168.0.64/smbcompartido /mnt/mipuntodemontaje
Comando Descripción
mount -t cifs Comando para montar un sistema de archivos de red CIFS (Common Internet File System).
-o username=fernando,password=alonso Opciones del comando, especificando el nombre de usuario (fernando) y la contraseña (alonso) para acceder al recurso compartido.
//192.168.254.44/smbcompartido Ruta de la red al recurso compartido: dirección IP (192.168.254.44) y nombre de la carpeta compartida (smbcompartido).
/mnt/mipuntodemontaje Punto de montaje en el sistema de archivos local donde se montará el recurso compartido.


Desconectar punto de montaje:

umount /mnt/mipuntodemontaje

Desconectar un punto de montaje de forma alternativa:

Para esta operación, es necesario tener instalado fuse3.

apt install fuse3

Luego, utiliza el siguiente comando para desconectar el punto de montaje:

fusermount -uz /mnt/mipuntodemontaje

4.1.2 Nautilus (Gestor de archivo de ubuntu)

Presionar CTRL+L y teclear en la línea de direcciones:

smb://192.168.254.44/smbcompartido


Nota: CTRL+L es el buscador de nautilus

4.2 Acceso desde Windows

4.2.1 Línea de Comandos

Conectar carpeta compartida con una letra de unidad usando:

C:\Users\antoniorodas>net use G: \\192.168.254.44\smbcompartido alonso /user:fernando

Tabla explicativa:

Comando Descripción
net use Comando para conectar o desconectar recursos compartidos de red en sistemas operativos Windows.
G: Letra de unidad que deseas asignar a la conexión de red (en este caso, G:).
\\192.168.254.44\smbcompartido Ruta de acceso al recurso compartido de red: dirección IP (192.168.254.44) y nombre de la carpeta compartida (smbcompartido).
alonso Nombre del usuario que tiene permiso para acceder al recurso compartido en la computadora remota.
/user:fernando Nombre de usuario con el que se inicia sesión en la computadora remota (fernando).

4.2.2 Explorador de archivos

Teclear en la línea de direcciones:

\\192.168.254.44\smbcompartido

Nota: No usar el explorador de Internet para acceder a recursos compartidos de SAMBA.
